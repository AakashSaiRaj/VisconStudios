Selected Project

Rupiq: Your Personal CFO

Rupiq is an offline-first Android and iOS app that helps users track spending, detect risk patterns, and make stronger financial decisions without sending sensitive data to external servers.

Rupiq is currently in alpha testing (invite-only). Public launch is planned for the first week of May 2026.

  • Local-first SQLite data model for transactions and goals.
  • Rule-driven insights for overspending and savings opportunities.
  • Privacy controls such as app lock and encrypted storage paths.
  • Future-ready architecture for privacy-safe AI augmentation.

Engineering Approach

How Viscon Studios builds software

Privacy by Default

Core logic stays on-device first. Data export and sync are opt-in, explicit, and user-controlled.

Incremental Intelligence

Start with deterministic rule engines, then introduce AI only when it improves outcomes without exposing raw user data.

Fast Product Loops

Ship MVPs early, test in real usage, and iterate with measurable improvements in reliability and clarity.

Delivery Roadmap

Execution sequence used for Rupiq

  1. Phase 0 Setup: project foundation, navigation, local DB schema.
  2. Phase 1 MVP: expense flows, categories, and early insight engine.
  3. Phase 2 Usability: better dashboard, health score, local notifications.
  4. Phase 3+ Goals, subscription detection, predictive advisory, optional AI.
